Консольді енгізу және шығару

Консоль шығысы

Консольге ақпаратты басып шығару үшін кірістірілген print() функциясы пайдаланылады. Бұл функция шақырылған кезде шығыс мәні жақша ішінде оған беріледі:

print ("Сәлеметсіз бе Студент")

Бұл код консольге "Hello Студент" жолын басып шығарады.

Бұл функцияның айрықша ерекшелігі әдепкіде ол жолдағы мәнді өздігінен шығарады. Мысалы:

print ("Сәлем әлем")
print ("Hello Студент")
print ("Hello Python")

Мұнда print() функциясына үш шақыру кейбір хабарламаны басып шығарады. Сонымен қатар, консольге шығарған кезде әрбір хабарлама бөлек жолға орналастырылады:

сәлем әлем
Сәлем Студент
сәлем Python

Бұл әрекет әрқашан қолайлы бола бермейді. Мысалы, біз барлық мәндердің бір жолда көрсетілетінін қалаймыз. Ол үшін біз end параметрін пайдаланып функция әрекетін теңшеуіміз керек. Бұл опция шығыс жолының соңына қосылатын және таңбаларды көрсетеді. end параметрімен print() функциясының шақыруы келесідей болады:

print(мән, end = соңы_таңбалар)

Әдепкі бойынша, соңы келесі жолға тасымалдауды көрсететін "\n" таңбасына тең. Сондықтан print функциясы әдепкі бойынша оған берілген мәнді бөлек жолға басып шығарады.

Енді функция келесі жолға аударылмайтынын, бірақ мәнді сол жолда көрсететінін анықтайық:

print("Сәлем әлем", end=" ")
print("Hello Студент", end=" ")
print ("Hello Python")

Яғни, енді көрсетілген мәндер бос орынмен бөлінеді:

Hello World Hello Студент Hello Python

Сонымен қатар, бұл бір таңба емес, таңбалар жиыны болуы мүмкін:

print("Hello World", end=" және ")
print("Hello Студент", end=" және ")
print ("Hello Python")

Бұл жағдайда шығыс хабарлар " және " таңбаларымен бөлінеді:

Hello World және Hello Студент және Hello Python

Консольді енгізу

Консольге шығарумен қатар біз консольден пайдаланушы енгізуін қабылдай аламыз, кіріс деректерін аламыз. Ол үшін Python input() функциясын анықтайды. Бұл функцияға енгізу сұрауы жіберіледі. Және біз енгізілген нәтижені айнымалыға сақтай аламыз. Мысалы, пайдаланушыға атау енгізу үшін кодты анықтайық:

name = input("Есіміңізді енгізіңіз: ")
print(f"Сіздің атыңыз: {name}")

Бұл жағдайда input() функциясына "Есіміңізді енгізіңіз:" жолы түріндегі енгізу сұрауы беріледі. Функция нәтижесі – пайдаланушы енгізуінің нәтижесі атау айнымалысына беріледі. Содан кейін print() функциясы арқылы осы айнымалының мәнін консольге басып шығара аламыз. Код мысалы:

Есіміңізді енгізіңіз: Евгений
Сіздің атыңыз: Евгений

Бірнеше енгізуі бар басқа мысал:

name = input("Сіздің атыңыз: ")
age = input("Сіздің жасыңыз: ")
print(f"Аты: {аты} Жасы: {жас}")

Бағдарлама мысалы:

Сіздің атыңыз: Том
Сіздің жасыңыз: 37
Аты: Том Жасы: 37

Барлық енгізілген мәндер str мәндері, яғни жолдар ретінде қарастырылатынын ескеріңіз. Жоғарыдағы кодтағы екінші жағдайдағыдай санды енгізсек те, Python енгізілген мәнді сан ретінде емес, жол ретінде қарастырады.
