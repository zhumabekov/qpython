Айнымалылар және деректер түрлері

Айнымалылар деректерді сақтауға арналған. Python тіліндегі айнымалы атауы әліпбилік таңбадан немесе астын сызудан басталуы керек және әріптік-сандық таңбалар мен астын сызу болуы мүмкін. Сонымен қатар, айнымалының аты Python тілінің кілт сөздерінің атауымен бірдей болмауы керек. Кілт сөздер көп емес, оларды есте сақтау оңай:

False   await     else     import    pass
None    break     except   in        raise
True    class     finally  is        return
and     continue  for      lambda    try
as      def       from     nonlocal  while
assert  del       global   not       with
async   elif      if       or        yield

Мысалы, айнымалыны анықтайық:

name = "Tom"

Мұнда name айнымалысы анықталған, және өзінде "Tom" сөзін сақтайды.

Python бағдарламалау тілінде айнымалыларды атаудың екі түрі қолданылыды: camel case және underscore notation.

Camel case айнымалы атауындағы әрбір жаңа ішкі сөз бас әріптен басталатынын білдіреді. Мысалы:

userName = "Tom"

Underscore notation айнымалы атаудағы ішкі сөздердің астын сызу арқылы бөлінгенін білдіреді. Мысалы:

user_name = "Tom"

Сонымен қатар регистр сезімталдығын ескеру қажет, сондықтан name және Name айнымалылары екі түрлі айнымалылар ретінде саналады.

# екі түрлі айнымалы
name = "Tom"
Name = "Tom"

Айнымалы мәнді анықтағаннан кейін біз оны бағдарламада пайдалана аламыз. Мысалы, кірістірілген print функциясын пайдаланып оның мазмұнын консольге басып шығарып көріңіз:

name = "Tom" # name айнымалысын анықтау
print(name) # name айнымалысының мәнін консольға шығару

Айнымалының айрықша ерекшелігі - бағдарлама барысында оның мәнін өзгертуге болады:

name = "Tom" # айнымалы атауы "Том"
print(name) # шығыс: Том
name = "Bob" # мәнді "Bob" деп өзгертіңіз
print(name) # шығыс: Bob

Дерек түрлері

Айнымалы деректер түрлерінің бірінің деректерін сақтайды. Python-да көптеген әртүрлі деректер түрлері бар. Бұл жағдайда біз ең негізгі түрлерін ғана қарастырамыз: bool, int, float, complex және str.

Логикалық мәндер

bool түрі екі логикалық мәнді білдіреді: True (рас) немесе False (жалған). True мәні бір нәрсенің ақиқат екенін көрсету үшін қолданылады. Ал False мәні, керісінше, бір нәрсенің жалған екенін көрсетеді. Осы түрдегі айнымалылардың мысалы:

isMarried = False
print(isMarried) # False

isAlive = True
print(isAlive) # True

Бүтін сандар

Int түрі 1, 4, 8, 50 сияқты бүтін санды білдіреді. Мысал

age = 21
print("Жасы:", age) # Жасы: 21

count = 15
print("Саны:", count) # Саны: 15

Әдепкі бойынша стандартты сандар ондық сандар ретінде қарастырылады. Бірақ Python екілік, сегіздік және он алтылық сандарды да қолдайды.

Санның екілік жүйені білдіретінін көрсету үшін санның алдына 0b префиксі қойылады:

a = 0b11
b = 0b1011
c = 0b100001
print(a) #3 ондық
print(b) # 11 ондық
print(c) # 33 ондық

Санның сегіздік жүйені білдіретінін көрсету үшін санға 0o префиксі қойылады:

a = 0o7
b = 0o11
c = 0o17
print(a) # 7 ондық
print(b) # 9 ондық
print(c) # 15 ондық

Сан он алтылық жүйені білдіретінін көрсету үшін санға 0x префиксі қойылады:

a = 0x0A
b = 0xFF
c = 0xA1
print(a) # 10 ондық
print(b) # 255 ондық
print(c) # 161 ондық

Консольге шығару үшін қандай жүйеде санды print функциясына берсек те, ол әдепкі бойынша ондықпен шығарылатынын атап өткен жөн.

Бөлшек сандар

Қалқымалы нүкте түрі 1,2 немесе 34,76 сияқты өзгермелі нүкте санын білдіреді. Нүкте бүтін және бөлшек бөліктер арасындағы бөлгіш ретінде пайдаланылады.

height = 1,68
pi = 3,14
weight = 68.
print (height) # 1,68
print(pi) # 3.14
print (weight) # 68.0

Жылжымалы нүкте нөмірін ғылыми белгілерде анықтауға болады:

x = 3,9e3
print(x) # 3900.0
x = 3.9e-3
print(x) # 0,0039

Қалқымалы санда тек 18 маңызды таңба болуы мүмкін. Сонымен, бұл жағдайда тек екі таңба пайдаланылады - 3.9. Ал егер сан тым үлкен немесе тым аз болса, онда көрсеткішті пайдаланып санды ұқсас белгілеуде жаза аламыз. Көрсеткіштен кейінгі сан негізгі санды көбейту керек 10 санының дәрежесін көрсетеді - 3,9.

Күрделі сандар

Күрделі тип күрделі сандарды нақты_бөлік+елес_бөлікj пішімінде білдіреді - елестетілген бөліктен кейін j жұрнағы келеді

complexNumber = 1+2j
print(complexNumber) # (1+2j)

Жолдар

str түрі жолдарды білдіреді. Жол "сәлем" және 'сәлем' сияқты жалғыз немесе қос тырнақшаға алынған таңбалар тізбегін білдіреді. Python 3.x жүйесінде жолдар Юникод таңбаларының жиынын білдіреді

message = "Сәлем әлем!"
print(message) # Сәлем әлем!

name = 'Том'
print(name) # Том

Сонымен бірге, жолда көп таңба болса, оны бөліктерге бөліп, кодтың әртүрлі жолдарына орналастыруға болады. Бұл жағдайда бүкіл жол жақшаға алынады, ал оның жеке бөліктері тырнақшаға алынады:

text = ("Laudate omnes gentes laudate"
"Магнификат в секуля")
print (text)

Егер біз көп жолды мәтінді анықтағымыз келсе, онда мұндай мәтін үш қос немесе жалғыз тырнақшаға алынады:

'''
Бұл түсініктеме
'''
text = '''Лаудаттық барлық құрметті құрметтер
Дүниедегі керемет
Ет анима мен мақтау
Дүниедегі керемет
'''
print (text)

Үштік жалғыз тырнақшаларды пайдаланған кезде, оларды түсініктемелермен шатастырмаңыз: егер үштік жалғыз тырнақшадағы мәтін айнымалыға тағайындалса, бұл түсініктеме емес, жол болып табылады.

Кірістірілген шығу реттері

Жолда бірнеше арнайы таңбалар болуы мүмкін - шығу реті. Олардың кейбіреулері:

  1. \: жолдың ішіне қиғаш сызық қосуға мүмкіндік береді
  2. \': жолдың ішіне жалғыз тырнақша қосуға мүмкіндік береді
  3. \": жолдың ішіне қос тырнақша қосуға мүмкіндік береді
  4. \n: жаңа жолға үзіледі
  5. \t: қойынды аялдамасын қосады (4 шегініс)

Бірнеше ретті қолданайық:

text = "Хабар:\n\"Hello World\""
print (text)

Бағдарламаның консольдық шығысы:

Хабарлама:
"Сәлем Әлем"

Мұндай реттілік кейбір жағдайларда бізге көмектесуі мүмкін, мысалы, жолға тырнақшаны қою, кестені жасау, басқа жолға орау. Бірақ олар да кедергі келтіруі мүмкін. Мысалы:

path="C:\python\name.txt"
print (path)

Мұнда path айнымалысы файлға кейбір жолды қамтиды. Дегенмен, жолда "\n" таңбалары бар, олар шығу реті ретінде түсіндіріледі. Осылайша біз келесі консоль шығысын аламыз:

C:\python
ame.txt

Бұл жағдайды болдырмау үшін жолдың алдына r таңбасы қойылады

path = r"C:\python\name.txt"
print (path)

Мәндерді жолға кірістіру

Python басқа айнымалылардың мәндерін жолға ендіруге мүмкіндік береді. Ол үшін жолдың ішінде айнымалылар {} бұйра жақшаға, ал f: символы бүкіл жолдың алдына қойылады.

userName = "Tom"
userAge = 37
user = f"name: {userName} age: {userAge}"
print(user) # name: Tom age: 37

Бұл жағдайда userName айнымалысының мәні {userName} орнына енгізіледі. Сол сияқты, {userAge} орнына userAge айнымалысының мәні енгізіледі.

Динамикалық тип тағайындау

Python - динамикалық типтелген тіл. Бұл айнымалы белгілі бір түрге қатаң түрде байланысты емес дегенді білдіреді.
Айнымалының түрі оған тағайындалған мән негізінде анықталады. Сонымен, қос немесе жалғыз тырнақшадағы жолды тағайындағанда, айнымалы str типінде болады. Бүтін санды тағайындау кезінде Python автоматты түрде айнымалының түрін int ретінде анықтайды. Айнымалыны қалқымалы нысан ретінде анықтау үшін оған ондық бөлшек ондық бөлшек болып табылатын бөлшек сан тағайындалады.
Сонымен қатар, бағдарламаның жұмыс істеу процесінде біз айнымалының түрін оған басқа типтегі мән беру арқылы өзгерте аламыз:

userId = "abc" # тип str
print(userId)

userId = 234 # тип int
print(userId)

Кірістірілген type() функциясын пайдаланып, айнымалының ағымдағы типін динамикалық түрде білуге ​​болады:

userId = "abc" # тип str
print(type(userId)) #

userId = 234 # тип int
print(type(userId)) #
