Бағдарламаларды жазуға кіріспе

Python бағдарламасы нұсқаулар жинағынан тұрады. Әрбір нұсқау жаңа жолға орналастырылады. Мысалы:

print(2 + 3)
print ("Сәлеметсіз бе")

Шегініс Python тілінде үлкен рөл атқарады. Дұрыс емес шегініс - бұл қате. Мысалы, келесі жағдайда қате пайда болады, бірақ код жоғарыдағымен бірдей дерлік болады:

print(2 + 3)
print ("Сәлеметсіз бе")

Сондықтан жаңа нұсқауларды жолға бірінші қойған дұрыс. Бұл Python және C# немесе Java сияқты басқа бағдарламалау тілдерінің арасындағы маңызды айырмашылықтардың бірі.

Дегенмен, кейбір тіл конструкциялары бірнеше жолдан тұруы мүмкін екенін ескерген жөн. Мысалы, шартты болса:

if 1 < 2:
print ("Сәлеметсіз бе")

Бұл жағдайда, егер 1 саны 2-ден аз болса, "Сәлеметсіз бе" жолы көрсетіледі. Мұнда қазірдің өзінде шегініс болуы керек, өйткені print("Hello") операторы өздігінен емес, if шартты құрылымының бөлігі ретінде пайдаланылады. Сонымен қатар, кодты пішімдеу нұсқаулығына сәйкес шегініс 4-ке еселік (яғни 4, 8, 16 және т.б.) болатын бос орындардың осындай санынан жасалған жөн. , содан кейін бағдарлама да жұмыс істейді.

Мұндай конструкциялар соншалықты көп емес, сондықтан қай жерде қажет және қай жерде бос орындар қою қажет емес екендігі туралы көп шатасуға болмайды.

Регізгі регистр

Python регистрді ескеретін тіл, сондықтан print және Print немесе PRINT өрнектері әртүрлі өрнектерді білдіреді. Ал егер консольге шығару үшін print әдісінің орнына Print әдісін қолдануға тырысамыз:

Print ("Сәлем әлем")

онда біз табысқа жете алмаймыз.

Пікірлер

Пікірлер код бөлігінің не істейтінін белгілеу үшін пайдаланылады. Бағдарлама аударылған және орындалған кезде интерпретатор түсініктемелерді елемейді, сондықтан олар бағдарламаның жұмысына әсер етпейді. Python тіліндегі түсініктемелер блокта және кірістірілген түсініктемелерде келеді.

Жолдық түсініктемелердің алдында фунт белгісі қойылады - #. Оларды бөлек жолға қоюға болады:

# Консольге шығару
# Сәлем әлемі хабарлары
print ("Сәлем әлем")

# белгісінен кейінгі таңбалардың кез келген жиыны түсініктемені білдіреді. Яғни, жоғарыдағы мысалда кодтың алғашқы екі жолы түсініктемелер болып табылады.

Олар сонымен қатар нұсқаулар орындалғаннан кейін тіл нұсқауларымен бір жолда пайда болуы мүмкін:

print("Hello World") # Хабарламаны консольге басып шығару

Пікірлерді блоктауда үш жалғыз тырнақша түсініктеме мәтінінің алдына және соңына қойылады: '''түсініктеме мәтіні'''. Мысалы:

'''
Консоль шығысы
Сәлем әлем хабарлары
'''
print ("Сәлем әлем")

Негізгі функциялар

Python бірқатар кірістірілген функцияларды қамтамасыз етеді. Олардың кейбіреулері өте жиі қолданылады, әсіресе тілді үйренудің бастапқы кезеңдерінде, сондықтан оларды қарастырайық.

Консольге ақпаратты басып шығарудың негізгі функциясы print() функциясы болып табылады. Біз шығарғымыз келетін жол осы функцияға аргумент ретінде жіберіледі:

print("Hello Python")

Егер бізге бірнеше мәндерді консольге басып шығару қажет болса, онда біз оларды үтірмен бөлінген print функциясына бере аламыз:

print("Толық аты:", "Том", "Смит")

Нәтижесінде барлық берілген мәндер бос орындармен бөлінген бір жолға жабыстырылады:

Толық аты-жөні: Том Смит

Егер print функциясы шығаруға жауапты болса, input функциясы ақпаратты енгізуге жауап береді. Бұл функция енгізу сұрауын қосымша параметр ретінде қабылдайды және біз айнымалы мәнде сақтай алатын енгізу жолын қайтарады:

name = input("Атын енгізіңіз: ")
print("Сәлем", name)
